# Anthropic Certification Partnership

Anthropic accepted StepUp.One's partnership request, requiring 10 team members to pass 4 certification exams.

## Requirements

- **10 people** must be certified
- **4 exams** to pass
- Must use **StepUp.One email** (not personal email) for proper credit and partner recognition
- Exams involve watching video content and completing quizzes

## The Four Courses

1. **Agent Skills** -- the immediate priority. Zero coding required. Listed as 10 hours but can be completed in ~3 hours once foundational concepts are understood. This is the most essential course and will be used daily.
2. **Building with Claude API** -- requires some technical understanding of APIs
3. **MCP (Model Context Protocol)** -- relatively easy
4. **[Fourth course]** -- relatively easy

## Key Concepts

### Agentic Skills
A skill is a set of ordered activities that an AI agent executes to complete a task. Example: a "Write a Blog" skill would involve 20 sequential steps. The term "agentic" is recent (emerged within the last month of the discussion), now used by Google, OpenAI, and Anthropic. Skills can be created, shared, and stored in cloud environments.

### Cost Awareness
- **Opus** is significantly more expensive than **Sonnet** (nearly 100x more than Haiku)
- Claude's API is 15x more expensive than comparable services
- Team must manage costs carefully when using premium models

## AI Access Strategy

Mohamed Anis proposed a unified premium AI access solution: integrating Claude's API into the existing StepUp.One platform so all team members can access the best models. The alternative of everyone subscribing individually ($20/mo each) was discussed but the API integration was preferred for cost management.
